# Proyecto_Movilidad_MoiraBittner
Transformar archivo crudo de movilidad a un reporte limpio y profesional. 

Este proyecto busca analizar los datos de movilidad de Chile del dataset Global Mobility Report, como parte del Taller Práctico de la asignatura Programación para la Ciencia de Datos.
El objetivo es aplicar un flujo profesional de datos: desde la limpieza y transformación, hasta la visualización y versionado con Git.
